# todo-list-app
A simple and modern task management web application designed to help users stay organized, track their tasks, and improve productivity.

## Features

- User-friendly landing page with login and sign-up UI
- Add, complete, undo, and delete tasks
- Circular progress bar to track task completion
- Notes section for quick reminders
- "Today", "Completed Tasks", and "Notes" menu system
- Dark mode support for better user experience
- Clean and responsive interface

## About the Project

This project was built as a personal learning project to improve my web development skills. It focuses on practicing DOM manipulation, UI design, and building interactive features using JavaScript.

## Tech Stack

- HTML
- CSS
- JavaScript

## Purpose

The goal of this project is to practice building real-world features such as task management, state handling, and dynamic UI updates while improving frontend development skills.
